<h1>Pathfinding Visualizer</h1><br />
This is a simple tool that lets you visualize different pathfinding algorithms(Currently just bfs).
You have a grid layout with n * m nodes. Start node is marked green and the end node is marked red.<br/>
This app can be run at: <a href="https://a-REAPER.github.io/pathfinding-visualizer/" target="_blank">https://a-REAPER.github.io/pathfinding-visualizer/</a>
